# AI-Based Automated UML Diagram Generator

This project automatically generates UML diagrams from Software Requirement Specification (SRS) documents using Natural Language Processing (NLP).

## Features
- Preprocess SRS documents
- Perform syntactic and semantic analysis
- Identify UML components
- Generate UML diagrams using PlantUML
- Classify relationships between components
- Allow human correction

## Technologies Used
- Python
- spaCy (NLP)
- PlantUML
- VS Code
- GitHub

## Project Structure

data/ – contains sample SRS document  
src/ – contains Python scripts  
output/ – generated UML diagram  

## How to Run

1. Install required libraries
2. Run main.py
3. UML diagram will be generated in the output folder
